# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, this is the README for this folder.

Two important files:

- `config.json`: The changeset config
- `CHANGELOG.md`: The changelog for the packages in this repo

## Useful commands

- `pnpm changeset` — Create a new changeset
- `pnpm changeset version` — Bump versions and update changelogs
- `pnpm changeset publish` — Publish packages to npm
